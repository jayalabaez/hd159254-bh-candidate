# HD 159254 — Black Hole Candidate Study

**Gaia DR3 4061400381769067392 · HD 159254 · SB1**

A luminous single-lined spectroscopic binary with a high-mass dark companion
toward the Galactic Centre, identified in the Gaia DR3 non-single-star catalogue.

## Key Result

| Property | Value |
|----------|-------|
| Mass function f(M) | **1.51 M☉** (> Chandrasekhar) |
| NSS significance σ | **343.4** (69× threshold) |
| Period P | 620.0 ± 1.4 d |
| Eccentricity e | 0.006 ± 0.006 (circular!) |
| K₁ | 28.62 ± 0.08 km/s |
| Distance | ~2527 pc |
| G mag | 7.92 |
| RUWE | 2.32 (elevated) |

The mass function exceeds the Chandrasekhar limit, excluding a white dwarf
companion.  However, f(M) < NS ceiling (~3 M☉), so the BH classification
depends on the primary mass M₁.  The extremely luminous absolute magnitude
(M_G ≲ −4) suggests M₁ ≳ 3–5 M☉, which pushes the companion mass into
the mass gap or BH regime.

## Status

**Candidate — requires spectroscopic follow-up for confirmation.**
